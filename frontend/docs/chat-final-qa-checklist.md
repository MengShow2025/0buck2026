# 聊天最终版 QA 清单

## 1. 会话与列表
- [ ] 会话列表分组显示：`官方群`、`个人会话`
- [ ] 官方群包含：活动群、推介群、广告群、动态群
- [ ] 会话最后一条消息与时间能实时更新
- [ ] 打开会话后未读数清零

## 2. 私聊/群聊消息
- [ ] 文本消息可发送，状态从“送达”到“已读”变化
- [ ] 图片消息可发送并预览
- [ ] 文件消息可发送并显示文件名/大小
- [ ] 语音消息可录制并播放

## 3. 消息操作
- [ ] 任意消息可“引用”
- [ ] 引用后发送的消息包含被引用预览
- [ ] 自己消息 2 分钟内可“撤回”
- [ ] 撤回后显示“你撤回了一条消息”

## 4. 会话内工具
- [ ] 会话内搜索可按关键词过滤消息
- [ ] `+` 菜单可打开魔法口袋并触发附件发送

## 5. 群管理权限
- [ ] 普通成员仅可退群
- [ ] 管理员可成员管理（添加/移除）
- [ ] 群主可设/撤管理员、转让群主、关群
- [ ] 官方群不显示成员管理与关群入口

## 6. 通讯录与好友管理
- [ ] 新的朋友支持同意/拒绝
- [ ] 同意后进入通讯录老友列表
- [ ] 老友支持拉黑，黑名单支持解除

## 7. 稳定性回归
- [ ] 页面刷新后聊天记录仍存在（本地存储）
- [ ] 页面刷新后群角色、群成员、隐藏群状态保持
- [ ] `npm run build` 通过
